Android 开发系列 之环境搭建 及 运行第一个 APP

初步安装:

参考: 菜鸟教程 CSDN

注意,需要翻墙环境 才能正常下载 sdk.

此处 指明一个坑:

Error:Cause: dl.google.com:443 failed to respond

这个报错 总的来说 还是因为网络环境的不正常.

解决方案: 修改GradleScripts 中的 bulid.gradle 添加 阿里源

// Top-level build file where you can add configuration options common to all sub-projects/modules.
 
buildscript {
    repositories {
 
        // 添加阿里云 maven 地址         
        maven { url 'http://maven.aliyun.com/nexus/content/groups/public/' }
        maven { url 'http://maven.aliyun.com/nexus/content/repositories/jcenter' }
 
        // jcenter()
        google()
    }
    dependencies {
        classpath 'com.android.tools.build:gradle:3.2.1'
        // NOTE: Do not place your application dependencies here; they belong
        // in the individual module build.gradle files
    }
}
 
allprojects {
    repositories {
 
        // 添加阿里云 maven 地址         
        maven { url 'http://maven.aliyun.com/nexus/content/groups/public/' }
        maven { url 'http://maven.aliyun.com/nexus/content/repositories/jcenter' }
 
        // jcenter()
        google()
    }
}
 
task clean(type: Delete) {
    delete rootProject.buildDir
}

再点击 Sync with Gradle Files 一般就在你的右上角 倒数 三四个.

点击 运行按钮
